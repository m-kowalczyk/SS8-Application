# SS8-Application

### Running CdJava.java
1. Open terminal and navigate to the directory containing CdJava.java
2. Use **javac CdJava.java** to compile the program
3. Use **java CdJava** to run the program
4. Enter two values separated by a space, with the first value representing the current directory and the second value representing the new directory

### Running cdpython.py
1. Opern terminal and navigate to the directory containing cdpython.py
2. Use **python cdpython.py** to run the program
3. Enter two values separated by a space, with the first value representing the current directory and the second value representing the new directory
